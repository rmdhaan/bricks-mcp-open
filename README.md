## ⚙️ System requirements

Your computer must meet these basic standards to run the software smoothly:

## 🧱 What this software does

This tool acts as a bridge. It allows AI models to read and modify your Bricks Builder data. You save time by using natural language to perform tasks that normally require manual clicking.

The server includes tools for these categories:

- Page creation: Generate new structures.
- Templates: Apply or change existing designs.
- Style management: Update CSS classes and layout settings.
- SEO: Adjust metadata and headings.
- Content: Edit text and media elements directly.

## 🚀 Setting up the connection

After you install the program, you must link it to your website.

1. Open the Bricks MCP application on your desktop.
2. Navigate to the Settings tab inside the app.
3. Enter your WordPress website URL.
4. Provide your Bricks API key if prompted.
5. Save these changes.

The application now listens for commands. You can open your preferred AI tool, such as Cursor or Claude, and point it toward this server to start sending instructions.

## 🛠 Using the tools

The application comes with 105 total tools. You do not need to install them individually. They activate as soon as you connect your AI assistant. To perform a task, type a request into your AI window.

Example requests:

- Create a new hero section with a specific background color.
- List all templates in my current library.
- Update the font size on my contact page to 16 pixels.
- Save the current layout as a new template.

The AI interprets these sentences and uses the Bricks MCP tools to complete the work on your site.

## 🛡 Security and privacy

The software operates locally on your machine. It does not send your data to third-party servers unless you initiate a command through your AI assistant. Your WordPress credentials stay on your computer.

## ❓ Frequently asked questions

Do I need to be a developer?
No. This tool makes complex tasks possible through simple text commands.

Does this slow down my website?
No. The tools only run when you ask them to perform a specific action. They do not add extra code to your site that impacts visitor experience.

Can I use this with other plugins?
The tools target Bricks Builder specifically. They may not modify settings controlled by independent plugins.

What happens if a tool fails?
The application provides an error log. Check the log to see where the process stopped. Most issues relate to incorrect URL paths or missing permissions on your WordPress site.
